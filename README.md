# cool-interactive-vis

An interactive visualization exploring gender representation in the Museum of Modern Art's collection over time.

## Dataset
- Source: [MoMA GitHub Repository](https://github.com/MuseumofModernArt/collection)
- License: CC0 (Public Domain)
- Files: Artists.csv (15,000+ artists) and Artworks.csv (130,000+ artworks)

## Features
- Stacked bar chart showing male/female artwork counts by decade
- Interactive year range slider
- Medium filtering (paintings, sculptures, photographs)
- Tooltips with exact percentages and counts

## Technologies
- D3.js v7
- HTML/CSS/JavaScript
- Deployed on GitHub Pages
